Powerful!

JavaScript powers you to do many cool things:

  • Page : Manipulate DOM Elements.
  • User : Handle User Interactions.
  • Server : Communicate/Exchange Data.

With the Page

  • Modify display context.
  • Change page layout.
  • Change styles.

With the User.

  • Response to events, e.g. Click, Drag & Drop.
  • Send notifications when needed.

With the Server.

  • Broader you communication scope. Not limited for specific structured server.
  • Reduce unnecessary requests for better performance.
  • Update in the background for better user experience.
